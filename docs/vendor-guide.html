Vendor Guide

Every tool and service in the Investment Tracker stack — what it does, why we chose it, and what it costs

Tools & Services

Vendor Breakdown

⚙

Next.jsnextjs.org

React framework by Vercel for full-stack web apps

Free / Open Source Framework
What it does in our app: Handles both the frontend (React pages, components) and backend (Server Actions, API routes). Uses the App Router for file-based routing and server-side rendering.
Why we chose it: App Router architecture, Server Actions for mutations without separate API, SSR for performance, and the best developer experience for React apps.
Cost: Free and open source. No licensing fees.
🔌

TypeScripttypescriptlang.org

JavaScript with static type checking

Free / Open Source Language
What it does in our app: Adds type safety across the entire stack -- from Prisma models to React components to Server Actions. Catches bugs before runtime.
Why we chose it: Type safety across the full stack, better IDE support, catches errors at build time, and makes large-scale refactoring safe.
Cost: Free and open source.
🎨

Tailwind CSStailwindcss.com

Utility-first CSS framework for rapid UI development

Free / Open Source Styling
What it does in our app: All component styling -- the pipeline table, cards, modals, investor portal layout, and responsive design. Styles are applied directly via utility classes in JSX.
Why we chose it: Fast development speed, consistent design tokens, tiny production bundle (only ships used CSS), and great integration with shadcn/ui.
Cost: Free and open source.
■

shadcn/uiui.shadcn.com

Pre-built, accessible UI components you own

Free / Open Source UI Components
What it does in our app: Provides buttons, dialogs, selects, dropdowns, tabs, forms, and other UI primitives used across both the admin and investor portals.
Why we chose it: High quality accessible components, fully customizable (code lives in your project), no vendor lock-in, built on Radix UI primitives.
Cost: Free. Components are copied into your codebase.
◆

Prismaprisma.io

Next-generation ORM for TypeScript and Node.js

Free / Open Source ORM / Database
What it does in our app: Manages all 14 database tables. Defines the schema, generates a type-safe client, handles migrations, and provides an intuitive query API.
Why we chose it: Type-safe database queries that match our TypeScript stack, auto-generated client from schema, declarative migrations, and excellent developer experience.
Cost: Free and open source. Prisma Accelerate (optional caching) is paid but not required.
☁

Supabasesupabase.com

Managed PostgreSQL database + file storage + more

Freemium Database + Storage
What it does in our app: Hosts the PostgreSQL database (all 14 tables), provides file storage for PDFs (private bucket with signed URLs), and handles connection pooling via PgBouncer.
Why we chose it: Generous free tier, managed PostgreSQL without DevOps overhead, built-in file storage with security policies, and connection pooling included.
Cost: Free tier includes 500MB database and 1GB storage. Pro plan at $25/month for production workloads with more storage, backups, and no pausing.
▲

Vercelvercel.com

Hosting platform by the creators of Next.js

Freemium Hosting / Deploy
What it does in our app: Hosts the entire application -- both the admin and investor portals. Auto-deploys from GitHub on every push to main. Creates preview URLs for pull requests.
Why we chose it: Zero-config deployment for Next.js, automatic GitHub integration, preview deployments per PR, edge functions, and analytics built in.
Cost: Free tier for hobby projects. Pro at $20/month per member for teams, with more bandwidth and build minutes.
🔒

NextAuthnext-auth.js.org

Authentication library purpose-built for Next.js

Free / Open Source Authentication
What it does in our app: Manages user login sessions using JWT tokens. Implements the credentials provider (email + password), stores role and companyId in the session, and integrates with middleware for route protection.
Why we chose it: First-class Next.js integration, JWT sessions without a session database, easy role injection into tokens, and middleware-compatible for route guards.
Cost: Free and open source.
✉

Resendresend.com

Modern email API for developers

Freemium Email
What it does in our app: Sends investor invitation emails containing login credentials. When an admin invites a new investor, Resend delivers the welcome email with their username and temporary password.
Why we chose it: Modern developer-friendly API, great integration with Next.js and React for email templates, custom domain support, and reliable delivery.
Cost: Free tier includes 100 emails/day and 3,000/month. Pro at $20/month for higher volume and custom domains.
▦

TanStack Tabletanstack.com/table

Headless, type-safe table library for React

Free / Open Source UI / Data Table
What it does in our app: Powers the admin pipeline table -- the core view of the application. Provides sorting, filtering, column visibility, pagination, and inline editing without dictating the UI.
Why we chose it: Headless design lets us use our own Tailwind + shadcn/ui styling, full TypeScript support, and handles complex table interactions like multi-column sorting.
Cost: Free and open source.
✓

React Hook Form + Zodreact-hook-form.com

Performant forms with schema-based validation

Free / Open Source Forms / Validation
What it does in our app: Handles all form state (asset creation, user management, investor invites, stage editing). Zod schemas validate inputs on both the client side (forms) and server side (Server Actions) using the same schema definitions.
Why we chose it: Minimal re-renders for performant forms, Zod provides runtime type validation that mirrors our TypeScript types, and shared schemas eliminate validation drift between client and server.
Cost: Free and open source.
🐙

GitHubgithub.com

Source code hosting and version control platform

Free Source Control
What it does in our app: Hosts all source code, tracks changes via Git, and triggers Vercel deployments on push. Pull request workflows for code review before merging to production.
Why we chose it: Industry standard for source control, native Vercel integration for auto-deploys, PR review workflows, and free private repositories.
Cost: Free for private repos and small teams.
📦

Dockerdocker.com

Container platform for consistent development environments

Free Development
What it does in our app: Runs a local PostgreSQL instance for development. One docker compose up command spins up a database identical to production.
Why we chose it: Eliminates "works on my machine" issues, provides a consistent local database environment, and makes onboarding new developers trivial.
Cost: Free. Docker Desktop is free for small teams and personal use.
🔐

bcryptjsnpmjs.com/package/bcryptjs

Pure JavaScript bcrypt password hashing

Free / Open Source Security
What it does in our app: Hashes user passwords before storing them in the database. Verifies passwords during login without ever storing plain text credentials.
Why we chose it: Industry standard bcrypt algorithm, pure JavaScript (no native dependencies), works in all environments including Vercel serverless functions.
Cost: Free and open source.
